COORDINATORAGENT: STRATEGIC DECISION-MAKING

IDLE State

  • Waits for events from SensorAgent
  • Transitions to PROCESSING on receipt

PROCESSING State

  • Analyzes threat severity (1-10)
  • Calculates optimal strategy
  • Transitions to ASSIGNING

ASSIGNING State

  • Creates task assignments
  • Sends deployment orders
  • Transitions back to IDLE
Decision time: <2 seconds

Decision Matrix

Severity Action
1-3 Standby Mode
4-6 Standard Rescue + Dome Monitor
7-9 Large Rescue + Dome Active
10 Maximum Response
