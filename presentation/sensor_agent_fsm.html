SENSORAGENT: CONTINUOUS EVENT DETECTION

IDLE State

  • Waits for activation command
  • Initializes monitoring systems
  • Transitions to MONITORING

MONITORING State

  • Polls news APIs and user reports
  • Checks for terrorist attack events
  • Transitions to ALERT_TRIGGERED

ALERT_TRIGGERED State

  • Creates alert message with details
  • Sends alert to CoordinatorAgent
  • Transitions back to MONITORING
IDLE
↓
MONITORING
↓
ALERT_TRIGGERED
↺
Response Time: 2-3s from detection to alert
